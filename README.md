# Sentiment Analysis using SentiWordNet

## Description
This project demonstrates a lexicon-based sentiment analysis method
using WordNet synsets and SentiWordNet polarity scores to analyze text sentiment.

## Methodology
- Text preprocessing and tokenization
- Lemmatization using WordNet
- Polarity scoring using SentiWordNet
- Rule-based sentiment classification

## Tools
- Python
- NLTK
- WordNet / SentiWordNet

## Academic Context
This work was developed as part of Master's coursework in Natural Language Processing.
